Network Infrastructure of Erovoutika

Interconnected Multi-Branch Network Infrastructure
This project demonstrates a fully functional multi-branch network simulation using Cisco Packet Tracer. The setup connects the Makati Main Branch, Manila Branch, and Dubai Branch through routers configured with RIP dynamic routing protocol to enable seamless inter-branch communication.

The Makati Main Branch serves as the core site, equipped with a firewall (ASA 5506-X), core router, distribution switches, and multiple access switches supporting various VLANs for users, servers, and WiFi. The Manila and Dubai branches feature their own routers, switches, VLAN segmentation, and wireless connectivity, allowing employees to securely access network services across locations.

This design highlights:
  • VLAN segmentation for efficient traffic management.
  • Dynamic routing (RIP) for branch connectivity.
  • DHCP server integration for automated IP assignment.
  • Firewall protection at the core network.
  • Scalable architecture supporting wired and wireless users.
  • Designed secure LAN/WAN topologies for academic project
  • Configured routers, switches, and others
  • Implemented VLANs, network segmentation, and access controls
  • Experience with network simulation tool (Cisco Packet Tracer)
