# Jessan Mojica Portfolio

This is a personal portfolio website showcasing my skills, projects, and professional experience in Information Technology, with a focus on network security, web development, and server administration.

## Features
- About Me section
- Downloadable resume
- Skills highlights (technical and soft skills)
- Projects gallery with detailed modals and images
- Network Infrastructure showcase with diagram and downloadable network file
- Contact and professional links

## Projects
- BIZCON: Business Management Platform
- ROBOLUTION: Collaborative Robotics Platform
- Online Enrollment System
- NutriWeb: Health & Fitness Website
- GWAPOAIRLINES: Flight Reservation System
- Arduino Based Project: Smart Dustbin
- BookingWebsite: Dining Reservation System

## Network Infrastructure
Includes a multi-branch network simulation using Cisco Packet Tracer, with downloadable .pkt file and a detailed diagram.

## How to Use
1. Clone or download this repository.
2. Place your images and downloadable files in the appropriate folders (e.g., `images/`, root directory).
3. Open `port.html` in a browser via a local server (e.g., XAMPP).
